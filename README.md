# HTML & CSS Learning Repository

A beginner-friendly repository for learning and practicing HTML and CSS fundamentals, including layout techniques and styling concepts.

## 📁 Contents

### HTML Files
- **index.html** - Basic HTML with inline CSS styling (borders, padding, box-shadow, color, and border-radius)
- **Flexbox.html** - Introduction to CSS Flexbox layout, demonstrating `justify-content`, `gap`, and flex properties
- **positing.html** - CSS positioning techniques and properties
- **zerodha.html** - Project/demo page implementation

### Assets
- **logo.svg** - Logo asset
- **landing.svg** - Landing page graphic asset

## 🎯 Topics Covered

- HTML Basics & Structure
- CSS Styling (colors, borders, padding, margins)
- Box Model Concepts
- Flexbox Layout
- CSS Positioning
- Responsive Design Basics

## 🚀 How to Use

1. Clone or download this repository
2. Open any `.html` file in your web browser to view the content
3. Inspect the HTML and inline CSS to understand the concepts
4. Modify the code and experiment with different CSS properties

## 📝 Notes

- Each file contains comments explaining CSS concepts and principles
- Inline styles are used for learning purposes; in production, use external CSS files
- Files include practical examples for understanding HTML/CSS fundamentals

## 💡 Key Concepts

- **Padding vs Margin**: Padding adds space inside an element, margin adds space outside
- **Flexbox**: A powerful layout tool for creating flexible and responsive designs
- **Box Model**: Understanding how elements are sized and spaced
