# Making PRD of Car Washing App

## Objective
 
 - Make an HTML website for a Car Washing service. Make it offline accessible
 
 ## Scope 
 - Build a basic skeleton containing the types of car wash and their respective prices
 - Display nearest car wash locations with service details 
 
 ## Features and Requirements 

 ### P1
 - User can see the types of car wash and their prices
 - User can see the nearest car wash shops
 - Display service availability and timings
 - Show customer reviews and ratings
 
 ### P2 
 - Add online booking system
 - Payment gateway integration
 - Send booking confirmations via email/SMS 


 ## User Story

 - User can easily browse different car wash types and prices
 - User can locate the nearest car wash shop
 - User can book a car wash service online and track the booking
 - User receives confirmations and updates about their service

 ## Technical Requirement 

 - HTML

 ## Design Requirement

 - Make the theme dark and vibrant
 - Use clear, readable fonts in high contrast colors
 - Add pictures of the different car wash types
 - Include location maps for nearby shops
 - Use intuitive icons for easy navigation
 
 ## Success Matrix

 - Users can easily book and view car wash prices
 - Show booking confirmations and real-time service updates
 - Achieve 90% customer satisfaction rating
 - Reduce booking time by 50%

 ## Timeline
 
 - 20 Days
